Controls for drone racing simulation game:
' ' - start 
W - forward 
A - left
S - backward
D - right 
UP arrow - gain altitude
DOWN arrow - drop altitude
RIGHT arrow - turn craft right
LEFT arrow - turn craft left 

1 - first-person camera
2 - default "easy cam" 
3 - 3rd person camera
t - tracking camera
4 - toggle altitude display 

c - toggle camera control on/off
ALT + mouse - move camera 
ALT + scroll - zoom in/out

f - toggle fullscreen 
o - display octree
v - toggle point display 
g - toggle GUI display 
b - toggle bounding box display

.ma files can also be found in geo folder
